## 💼 Experience

### 🏦 Data Scientist — Goldman Sachs *(Aug 2022 – Jul 2024)*

- 🔐 Architected a **real-time risk-scoring and fraud-detection platform** processing **10M+ daily transactions** using an XGBoost + Isolation Forest ensemble on AWS, delivering an estimated **\$50M+** in annual fraud-loss prevention while maintaining **99.2% precision**.
- 📄 Led **NLP and LLM development** for regulatory document analysis across **1M+ documents** using BERT, RoBERTa, and PyTorch, achieving **94% F1-score** and reducing manual compliance review workload by **35%**.
- ⚙️ Built **end-to-end MLOps pipelines** on Databricks and AWS SageMaker leveraging MLflow, Docker, Kubernetes, and CI/CD, reducing model deployment time by **60%** and standardizing monitoring across **20+ production models**.
- 💡 Designed a **deep-learning customer-churn prediction model** in TensorFlow and Keras, identifying high-risk customer cohorts **30 days in advance** and driving **\$12M** in annual retention savings.
- 🚨 Developed a **fraud-detection ensemble** (XGBoost + LightGBM) that reduced false-positive alerts by **35%** while maintaining **99.2% precision**.
- 🔍 Prototyped **RAG workflows** over internal regulatory corpora using vector embeddings (FAISS) and LLM-based summarization, accelerating analyst research cycles by an estimated **40%**.
- 🔄 Engineered automated **ETL pipelines** processing **50M+ records daily** using PySpark and Apache Airflow on the firm's cloud data lake.
- 🚀 Shipped a **production-grade credit-risk scoring REST API** using FastAPI and LightGBM with **sub-50ms p99 latency** and SHAP-based explainability for model-risk governance.
- 📊 Operationalized **model monitoring, drift detection, and automated retraining** schedules ensuring continued accuracy across all production models.
- 🎓 Mentored a team of **4 junior data scientists** on MLOps best practices, improving team productivity by **25%**; partnered with Risk, Compliance, and Engineering leadership aligned with **SR 11-7** model risk management standards.
